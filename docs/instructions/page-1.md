# Списание времени

Это пошаговая инструкция для списания времени в Jira.

1. Перейдите в соответствующую задачу.

2. Нажмите на кнопку **+** для добавления времени.

3. Заполните поля, указанные на рисунке.

4. После заполнения полей нажмите на кнопку **Работа**.
