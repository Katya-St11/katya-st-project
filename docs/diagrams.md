# Сценарий

## **Пользовательский сценарий «Просмотр заметки»**

*Действующие лица*:

1. Пользователь.

2. Приложение.

3. Бэк.

4. База данных.

## **Предварительные условия**

Пользователь должен находиться на главном экране.

## **Выходные условия**

На главном экране отображается список заметок.

## **Основной сценарий**

Пользователь нажимает на заметку 1.

Приложение отправляет запрос Бэку на открытие заметки 1 из Базы данных.

Приложение отображает окно с текстом заметки.

Пользователь просматривает заметку.

Пользователю доступны кнопки **Редактировать**, **Удалить**, **мои заметки**.
